# 🎮 프로젝트 기능 (완료 & 예정)

## ✅ 완료된 기능

### 게임 시스템
- [x] **싱글플레이** - 1인 게임, 즉시 플레이 가능
- [x] **난이도 선택** - 쉬움(12초) / 보통(8초) / 어려움(5초)
- [x] **타이머** - 0.1초 단위 정밀도
- [x] **점수 계산** - 단어길이×10 + 콤보보너스
- [x] **콤보 시스템** - 연속 정답 시 추가 점수
- [x] **HP 기반 생명** - 틀릴 때마다 1 감소
- [x] **제시 글자** - 48개 한글자 자동 생성
- [x] **사용 단어 추적** - 사용한 단어 중복 방지

### UI/UX
- [x] **다크 테마** - 눈 보호, 야간 게이밍 최적화
- [x] **모바일 반응형** - 모든 해상도 지원
- [x] **Framer Motion 애니메이션**
  - [x] 폭탄 떠다니기
  - [x] 폭발 이펙트
  - [x] 화면 진동
  - [x] 콤보 팝업
  - [x] 부드러운 페이드/슬라이드

### 데이터 & 랭킹
- [x] **단어 검증** - 300+ 한국어 기본 단어
- [x] **점수 저장** - 게임 종료 후 저장
- [x] **랭킹 조회** - 상위 20개 표시
- [x] **통계 표시** - 라운드, 콤보, 단어 수

### API & 통신
- [x] **REST API** - 단어 검증, 랭킹, 게임방
- [x] **Socket.io 구조** - 멀티플레이 기반 완성
